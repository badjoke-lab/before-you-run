# Candidate Sources Runbook

## Scope

Before You Run v0.2 candidate collection is not active yet. This runbook and related metadata are preparation only.

## Purpose

The source catalog defines public-safe metadata for future candidate discovery workflows.

## Source classes

- **Primary**: Authoritative advisories or official incident references used for stronger confirmation.
- **Reference**: Reputable research or reporting used for context and defensive awareness.
- **Signal**: Early or community signals used only for discovery and follow-up verification.

## Handling boundaries

- Signal-only items are not treated as confirmed.
- Candidate items must be verified and converted into safe "what to avoid" cards before publication.
- Published content must remain defensive and beginner-safe.
- No exploit steps or offensive operational detail should be published.
