# Tripwire v0.1 Release Notes

Tripwire v0.1 is the first MVP release candidate.

## Included

- Static English and Japanese pages
- 20 risky-action cards
- Search and filters
- In-page card detail view
- AI/Agent Safety copy actions
- Checklists
- Command safety reference
- After-incident guide
- Sources explanation
- JSON validation

## Not included

- Automated candidate collection
- Advisory database integration
- Social signal collection
- User submissions
