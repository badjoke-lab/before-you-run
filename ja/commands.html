コマンド安全リファレンス

よく使うコマンドでも、条件次第で大きな影響があります。

rm -rf

注意理由: 再帰削除で重要ファイルまで消える可能性があります。

先に確認: 実行場所、対象パス、バックアップ有無。

curl ... | sh

注意理由: 取得した内容をそのまま実行します。

先に確認: URL、配布元、内容を保存して確認すること。

sudo ...

注意理由: システム全体への変更になります。

先に確認: 管理者権限が本当に必要か。

chmod -R ...

注意理由: 権限を広く変えすぎることがあります。

先に確認: 対象範囲と最小権限の方針。

パッケージの install scripts

注意理由: 追加コマンドが自動実行される場合があります。

先に確認: メタ情報、scripts、公開元。

不明点がある場合は実行前に確認を求めてください。
